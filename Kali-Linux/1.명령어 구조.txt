part4. 파일 정보 확인

* ls [파일 or 디렉토리 경로] [options]
-- 만약에 전달이 인자가 없으면, 현재 경로를 넣어준다고 침

ls -l
-- 파일 전체와 파일 정보를 알려줌

ls -a
-- 숨김 파일까지 보여줌
리눅스에서 .(점)이 파일 이름 앞에 붙으면! 숨김 파일 인식

ls -a -l	=	ls -al	=	ls -la 등등
-- 숨김 파일의 모든 정보까지 알려줌

-rw-r--r-- 1 root root 321   Feb 22 6:57 /etc/passwd
	파일권한	     |크기| 수정 된 날짜| 	파일 이름  |

- : 파일
d : 디렉토리
