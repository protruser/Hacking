파일 시스템
>> 파일이 어떻게 저장되는지~!

* 트리 구조

/ -> root
		-> Desktop
		-> Download
	-> bin
	-> tmp

/bin -> 리눅스에서 사용하는 바이너리 파일들
/dev -> device, 리눅스 연결되는 하드웨어 장치 파일
/etc -> 리눅스의 설정 파일
/home -> 각 계정의 홈 디렉토리들이 존재하는 디렉토리
/lib -> library 약자, 공유 라이브러리 파일들이 존재하는 디렉토리	(.so -> 공유 라이브러리 파일들)'
/root -> root 계정의 홈 디렉토리
/sbin -> system binary, 시스템 관리를 위한 명령어(바이너리)
/tmp -> temp, 임시 디렉토리 -> 누구나 이 디렉토리를 쓸 수 있다.
					-> 껏다 키면 여기 있는 파일들이 다 사라짐!
/var -> 리눅스에서 사용하는 프로그램들이 사용하는 파일들을 보관
